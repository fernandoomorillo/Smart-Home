# Smart-Home

Se pedia realizar una página web en HTML y CSS sobre un Dashboard para una SmartHome. En la página se tienen que ver una serie de widgets, un apartado para poder ver la página en modo claro o modo oscuro y tener un MediaQuery para móveles, tablets o escritorio.

## Figma
Primero hacemos un figma para decidir como ibamos a estructurar la página web y decidimos la gama de colores.
[Figma](https://www.figma.com/design/B5hE7y45n6RSl5uyV6zwPW/SmartHome?node-id=0-1&t=ohRFO6uqxejsQZwL-1)

## Desarrollo en HTML y CSS
Cuando terminamos el figma, ahora pasamos todas las ideas a HTML y CSS, el desarrollo se realizó en Visual Studio Code.

## Herramientas y tecnologías utilizadas 
- Figma
- Visual Studio Code
- HTML
- CSS
